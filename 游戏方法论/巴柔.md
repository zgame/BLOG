# 抱腿

	（抱双腿）
	1. 把对方手臂拉高，突然下潜，一个膝盖着地， 抱对方的双腿膝窝，头顶住对方身体，手往一侧拉拽
	2. 抱对方的双腿膝窝后，后脚绕到前面勾住对方脚踝，注意自己的脚一定要脚底朝上，防止崴到自己的脚踝，身体往前压，腿往后拉

	（抱单腿）
	1. 下潜，一个膝盖着地， 双手抱对方的单腿，可以举起来，摔对方或者勾对方支撑脚； 也可以头靠住对方大腿内侧，双腿夹紧对方腿防止逃脱，肩膀下压
	2. 一手抓对方衣领， 另外一只手抓对方的袖子， 往后撤一步，目的是让对方上步，然后抓袖子的手放开去报对方的单腿
	3. 抱单腿，一手抱膝窝，一手抓脚踝，头贴住对方大腿然后往前下方向压，同时抓脚踝的手往后拉


# 摔

	（舍身技）
	抓衣领，衣袖，头靠近，半蹲，往后坐，伸腿绊倒对方
	
	(抱腿)
	抓衣领，衣袖，衣袖拉直，抄腿，转抱单腿或者直接后摔

	（背负投和腰技）
	1.背负投，拉直胳膊，肩顶住对方腋下，顶起，转身半蹲，腰贴抵住对方，拉手摔
	2.大腰，拉衣袖， 大领或搂腰， 转身半蹲撅屁股抵住对手胯部， 腰顶起对方重心，摔
	3.扫腰，拉衣袖， 大领或搂腰， 转身半蹲撅屁股抵住对手胯部，一脚着地， 一脚要用例去扫对方两条腿
	4.一本背负投，拉衣袖， 大领或搂腰， 转身一腿半蹲，一腿跪地，撅屁股抵住对手身体，双手拉住对方胳膊，往下拽， 对方摔之后，要用手去盖住对方，侧压
	5.拉衣袖， 大领或搂腰， 转身半蹲撅屁股抵住对手胯部，一脚着地， 如果对方距离远，一脚扫对方两条腿中间

	（袖钓入腰）
	站立双手抓对方衣袖，一手抓袖往对方的另外一面腋下绕，同时上步下蹲，腰靠住对方身体， 顶起摔

	（拿背）
	拉衣袖，换手，往测后方拉，另一手抓对方后背转拿背

	（足技）
	拉直胳膊，肩顶住对方腋下，顶起，上步同时扫腿

	(开放式防守)
	1.双手拉对手的同侧衣领， 用力往下拽， 把对方拉下来， 然后再用其它技巧
	2.抓对手衣领，往左侧移动， 左腿勾住对方右腿，顺势倒下， 转德拉防守

# 过腿

	（拉腿）
	1. 拉住对方脚踝，另外一只手拉住脚背， 往后拽，然后往一面扔
	2. 拉住对方裤脚， 把对方腿拉直， 往一侧扔
	3. 手肘抓住对方的脚踝，手搭扣，脚踩住对方的腿，然后肌肉切对方小腿
	4. 抓住对方双腿往高抬，往下压，然后往一侧倾斜，上膝盖压住，另外的对方腿要扛在我的肩上并且手抓住对方的衣服防止对方腿压我，另外一只手抄对方脑后，然后切膝过腿


	（压迫过腿）
	肩膀抵住一条腿，左手反抓对方的衣领，并把左手的手刀放到对手脖子位置，要压迫对方让对方不舒服，围绕对方的腿绕一下转测压，两个方向都可以过腿

	(斗牛士)
	1. 双手抓住对方的两个裤脚，往一侧上步，然后膝盖抵住对方腹部
	2. 双手抓住对方的两个裤脚，左手往下按，同时左肩去压住对方
	
	（切膝）
	一条腿在对方双腿内侧，一条腿在外侧，向外侧横向移动，然后切膝，切膝后立即抄对方腋下，用头去压平对方，抄好对方手臂。
	用膝盖压住对方的大腿，划过去转侧压,	在切膝的过程中，目的是将对方身体侧过来，注意一定要抢腋下，否则下位的人拿到腋下可以拿背， 抢到腋下之后，肩膀压住对方的肩膀， 头顶住对方的头，抄好对方的胳膊，再过腿, 在切膝的过程中，腿不能太近，容易被抱腿，抽腿的时候要转跨，屁股顶住对方，抽腿

	（侧向移动过腿）
	适合搞定德拉，蝴蝶，蜘蛛等
	身体往一侧移动，把对方身体侧过来，这样对方的防守就会降低，我的一条腿压在对方的腿上，滑膝过来

	（跳步过腿）
	用头压住对方，一手抓对方腰带或者衣服，一手抓对方裤子，然后跳步过腿


# 侧压

	（绞脖子）
	1.袖车, 绕脖子后面的左手抓住自己的右侧袖子，然后右手切喉咙，一起用力
	2.十字绞，侧压，一手抓后脑领子， 另外一手顺着后脑绕道另外一侧抓领子，近身膝盖压住对手胳膊，用力下压

	（借助道服）
	1.衣襟绞，测压左手抓后领，右手拿对方下侧的衣襟交给左手，然后右手从衣襟下面穿过去，切对方的脖子，形成挤压
	2.棒球绞，侧压，一手拿对方衣襟，包住对手胳膊递给穿后脑的手握住， 然后从对方手下穿过抓棒球把位，近身膝盖压住对手胳膊，用力下压

	（美国锁）
	测压的时候，用我的肩膀去帮忙压住对方的手臂，然后抓住对方手臂，再绕过头来做美国锁

	（手臂三角绞）
	手往头上伸，带动对方手臂往上，双手搭扣，然后跳跃过对方另外一侧，手互相抓自己的手肘，身体横移到十字方向

	（攻击手臂）	
	1. 左手抓后领，腿改袈裟， 然后迈腿压住对方的头开始降服，不能降服就起身转十字固
	2. 对方背对我， 拿住对方胳膊做木村，可以转十字固
	3. 对方正对我， 拿住对方胳膊往后倒，做木村，
	4. 做对方美国锁，对方手来帮忙，扳过来对方手，转手臂三角绞
	5. 起身，双手搭扣直接绞对方脖子，手穿到对方手臂下面，绕颈， 转手臂三角绞
	6. 做美国锁，对方手来帮忙， 直接起身做十字固
	7. 测压往上爬一点，一腿跪腹部，一腿绕到对方头上面，抄住对方的手臂，然后往后坐起来，把对方拉起来面对我侧身，直臂固，或者往两侧掰做美国锁和木村锁都可以
	8. 测压，把对方远端的手臂往上推，我左手抄对方脖子后面伸过去抓对方腋下，起身让对方侧身，我的脚立起来到对方脖子后面，对方的手肘在我脚后面，右手抵住对方身体防止他逃脱，我起身做对方胳膊，可以做直臂固，也可以右手抓自己左脚踝，往后坐同时加紧腿，做反向三角锁，然后拿对方胳膊，做木村，直臂固都可以

	

# 袈裟固

	（美国锁）
	右手在对方腋下位置，左手拉住对方手肘，右腿抵住对方的身体，用右手二头肌把对方手肘拿过来，然后左手抓住对方手腕，右手抓左手手腕，然后侧身做美国锁	

	（直臂固）
	 1. 把靠近对手的腿往前伸， 然后靠外的腿跨过对方的头，然后拧身，掰对方的胳膊
	 2. 袈裟固，右手搂脖子，左手搂住对方胳膊，把对方胳膊往左腿下面放可以直臂固，如果对方收回来，右腿从下面往上勾住对方手臂，身体胯往上顶
	
	（十字固）
	直接从袈裟位，跳对手身上，远侧腿上对方脖子， 做十字固

	（反三角锁）
	袈裟固，右手搂脖子并伸到自己右大腿这里，开始左腿做对方手臂直臂固，然后转身右手肘抵住对方身体，右腿膝盖挤到对方身体下面，左腿上来勾住对方头做反三角锁。

	
	

# 上位骑乘

	（控制）
	1. 上位骑乘的腿可以自己搭扣，防止对方腿勾我的腿，双手往上打开，可以更稳固
	2. 可以腿勾住对方的腿，上身压下去，一只手搂住对方脖子，然后把头和身体放到另外一侧保持平衡

	(连击)
	1. 双手按对方一个胳膊，一只手按手腕，一只手按手肘，压到地上，如果对方不反抗就可以做肩锁， 如果对方侧身过来，那么我就把重心转移到对手的后背，类似拿背，但是我的单脚还是跨过对方身上的，抓对方手腕的手就换抓对方脖子下面的对侧衣领，抓住衣领之后另外一只手从对方的腋下穿上来，然后切对方另外一侧脖领，类似拿背的片羽绞
	2. 首先做对方美国肩锁，对方另外一只手来帮忙，抄手肘转十字固， 十字固不行，把胸上的腿穿过来，抓住对方的同侧胳膊，拉起来对方转三角锁

	(攻击手臂)
	1. 上位骑乘十字固，单膝跪耳测，另外脚半蹲，侧身，抄手，十字固
	2. 单手拉起对方身体，让对方侧身，腿前后固定，抄手，坐肩膀，十字固
	3. 一手抱头，一手打开保持平衡，双腿逐渐网上压住对方手臂，一腿贴耳，一腿弯到环绕，抄手，往后顶胯，两侧都可以十字固
	4. 木村，抓住对手手腕做木村， 对方另外手过来帮忙就转十字固
	5. 上半身压住对方，逐渐往上走， 手和腿把对方的胳膊往上顶，抄胳膊和头，做直臂固，不行可以拉胳膊转十字固
	6. 对方双手抱紧，我一只手从下往上穿过去，然后往后走抱起对方胳膊，把对方提起来之后，我同侧的脚蹲起，跨过对方的头，用脚背抵住对方脖子，双手拉头




# 全防守

	（控制）
	拉对方下来，如果对方手靠外， 木村->断头->穿手，搭扣，翻，上位，不松搭扣，腿过来，折颈。
	拉对方下来，如果对方手靠内， 可以扫技，
	如果对方拉不下来，起身抱对方，抓衣领，拉下来，可以做扫技，或者扫技虾行后顶膝的时候，顶高一点顺势转到肩膀上三角扣直接下位降服
	如果对方要逃脱，双手去抓对方的一个衣袖，控制对方的一个手臂，尽量把他手臂拉直， 1.可以拉起来，头穿过胳膊，用身体压住对方胳膊，手抄住控制住 2 可以拉到头的位置，抄腋下做直臂固 3 可以拉到左侧做腿木村或者中途转三角锁 4 可以拉到右侧转拿背
	三角锁，挺胯，拿手转腿木村，再回来转十字固


	（转拿背）
	1. 反向木村把位，一只手抓住对方不同侧手腕， 另外一只手肱二头肌弯曲扣住对方手肘，侧身留出空间，一手拉直对方胳膊并且不要放，另外一只手拉对方后背，往后倒可以转拿背
	2. 全防守，抓衣领，衣袖，解扣，虾行，塞膝盖，扫，但是扫失败的时候，把脚插对方腿内做半防守，手抓另外的衣袖，用力拉向一侧，同时膝盖划走，为了拿背，膝盖半防守往外掰，分开对手的腿，这时候勾对方的腿来拿背，如果对方不给腿，就顶跨拉直对方，再勾腿，完成拿背。
	3. 一只手抓住对方不同侧手腕， 另外一只手肱二头肌弯曲扣住对方手肘，往一侧拽同时身体往反方向移动保证手臂挺直，脚脱扣抓对方背，在抓住控制住的对方的手，回到全防守位置，左手抓对方手腕的话，左脚蹬对方胯，右腿打腋下，右手抄对方腿扫倒，然后提一下对方身体让对方侧身，右手穿进来抓左手手腕，往后坐拿背



	（衣襟绞背后穿攻击脖子）
	1. 可以左手拿对方衣襟从后面穿过来，交给右手做棒球握，右手前臂抵到对方的喉咙， 如果对方来推我的右手肘，左手抓住对方衣袖拉过来，左手拉直并且往下推，同时右手用力下拉
	2. 左手拿对方的衣襟从后面穿过来，交给右手做棒球握，双腿下压让对方的头在我的胸前位置，左手交叉去抓衣襟或者对方肩膀，十字绞，如果对方推我的手肘不让我绞，那么转1

	（衣襟绞侧面穿攻击脖子和手臂）
	1. 左手拿对方的衣襟，虎口向上握住，衣襟从对方的肱三头肌位置拉过来，同时右手把对方的左手也拉到我的左侧，双腿向上台 为了让对方头部靠近我的胳膊，然后右手从后面绕过对方脖子去抓衣襟，抓住之后可以右手用力拽，左手按住对方胳膊；或者左手也同时在内侧向外推，形成挤压
	2. 左手拿对方的衣襟，虎口向上，衣襟从对方的肱三头肌位置拉过来，同时右手把对方的手也拉到一侧，左手抓衣襟的手不动去控制住对方胳膊，然后上腿做十字固
	

	(下位十字固，攻击手臂)
	1. 双手抓对方的衣袖，把对方的手拉直，对方的手肘在我的腰带处，一只手抓衣袖，另外一只手从对方手上面，我的手下面穿进来搭扣，注意这个时候如果对方想抽走手臂， 他会拉直手臂，这个时候就是上腿的时候，两条腿都要上来，一侧腿要越过头部，做下位十字固
	2. 双手抓对方的衣袖，把对方的手拉直，对方的手肘在我的腰带处，右手抓衣袖，左手从对方手上面，从右手下面穿进来搭扣，如果对方不拉直胳膊，那么我把对方拉下来，把抓住的胳膊交给从后面绕脖子的左手抓住，然后右手抄对方腋下，抄对手的左胳膊，左手也来一起抓住，然后左面侧身，右腿也拿上来，跪到对方的脖子处，开始对对手的左臂进行反向十字固
	3. 抓住对方衣领，越深越好，做十字固搭腿，手不放
	4. 下位十字固，拉对方手，一个腿打腋下，侧身，另外一个腿拿上来做十字固， 顶胯
	5. 把对方拉下来，左手搂住对方后脖颈，右手穿到对方手臂深处抓好，推头同时右腿上撩，左腿上来压对方近侧脖子形成十字固
	6. 把对方拉下来，左手搂住对方后脖颈，右手穿到对方手臂深处抓好，推头同时右腿上撩，左腿上来压对方远侧脖子并勾住右脚，肩膀十字固


	（直臂固，抄腋下）
	1. 双手抓对方的一个衣袖，把对方的手拉直，对方的手肘在我的腰带处，右手抓衣袖，左手从对方手上面，从右手下面穿进来搭扣，如果对方不拉直胳膊，那么我把对方拉下来，把抓住的胳膊交给从后面绕脖子的左手抓住，然后右手抄对方腋下，抄对手的左胳膊，用右手内侧手肘抵住对方手肘，对方的手要卡到我脖子的位置，左手过来帮忙拉我的右手肘，做直臂固
	2. 如果对方用双手抓裤子抵住直立，我方抓住对方一只手，另外手抓对方同样这只手的衣袖，拉出来，控制对方一只手，控制对方手之后可以抄腋下，然后双手搭扣，用头抵住对方手腕，做直臂固
	3. 全防守，抱住对方头，拉下来，抄腋下，双手搭扣，解腿扣，虾行为了侧身，电话锁对方胳膊，对方会弯曲胳膊来防守，这时候抓对手手腕往回掰，会硌到对方的前臂肌肉，不能降伏，往外侧掰，对手会失去重心，起桥转上位，如果手依然抓住，可以做美国锁，注意美国锁要拉到靠近对方身体再掰，记得掰对手的二头肌位置，不是前臂

	（肩胛固，反木村把位）
	右只手抓住对方右手腕往前顶， 另外左手肱二头肌弯曲扣住对方右手手肘往回拉，这样就比较紧，侧身留出空间，上右腿搭到对方脖子后面，形成肩臂锁，左腿蹬对方胯，右手一起来拉对方的手肘


	（下位十字绞，攻击脖子）
	1. 左手抄对手的手肘，从手肘下面穿过去，抓对方远端的衣领，右手抓另一侧已领，右手往前顶，左手往回拉，做十字绞
	2. 抓左侧，对方右侧的衣领，左手在上，右手在下，左手发力往左下带，对方的头下来之后，手腕转动绕过对方头，注意把位不能变，绕过之后就是十字绞
	3. 抓左侧，对方右侧的衣领， 右手在上，左手在下，左手发力，往左下带，对方的头下来之后，左手去抓对方的腰带，左手发力用胳膊推对方头，右手拉，形成挤压


	（下位三角锁，攻击脖子）
	1. 双手各自抓对方的衣袖，右手推，左手拉，同时挺胯，上右腿到对方肩膀上，左手反方向拉住对方胳膊，向右侧身，把右脚放到对方脖子后面，左脚蹬对方胯或者地面，有一定空间再搭扣，双腿膝盖加紧，挺胯
	2. 如果双脚已经上来，对方手藏起来不给拉手， 用双手在对方背后搭扣，然后双腿下压，双手勒紧
	3. 如果双脚已经上来，对方手藏起来不给拉手， 那么把三角锁反向，用左手抵住对方的胳膊肘，左腿被右腿压住搭扣，用力


	（肩膀三角扣，攻击手臂）
	肩膀十字固，拉对方手，一个腿打腋下，另外一个腿搭到对方肩上，脚搭扣，顶胯
	双臂锁 ， 拉对方双手，两腿夹住对方，一个腿肩上，一个腿在肩膀，顶胯
	三角扣十字固， 拉对方手，一个腿在肩上，一个腿在腋下，脚搭扣，往一侧掰对方手臂,也可以转木村
	肩锁，横向推对方的手肘，当手肘超过中线之后，开始双手掰对手的胳膊
	木村，对方的手背过去藏起来，不转腿木村，而是双手搭扣开始压按对方的肘关节
	压脖子，直接抓对方的衣领，然后用拳峰去挤压对方的脖子
	直臂固，抄腋下，然后做直臂固
	肩甲固，左手抓住对方左手腕，右手从对方手肘中穿过并按住自己的左大腿，把对方的手藏入腿缝，把左腿拿到对方肩上来，右手搭住大腿，另外一只手掰对方手肘；或者右腿蹬对方胯，同时右手掰对手的手肘



	（全防守扫技)
	1. 剪刀扫 ，全防守，解扣，顶膝，靠近，后倒，扫腿或者蹬腿
	2. 蝴蝶扫 ，全防守，解扣，胳膊夹住对方一只胳膊，另外一个手抄腰，脚勾到对方的大腿内侧，往侧方向倒，脚往上挑
	3. 木村扫 ，全防守，先要侧身，目的是把屁股挪一下，起身木村扫， 如果扫技失败，可以转三角锁（如果对方手向后撑地有空间），腿木村（对方手贴很紧），十字固（我能拿住对方的胳膊）
	4. 钟摆扫 ，全防守，手拉住对方后脑按下来，如果对方手在我胸前，那么给对方手肘塞进去压住，一只手压住对方后脖子，脱扣，侧身， 另外一只手抄对方的腿，然后我的抄腿手同侧的腿用力下压， 转上位
	5. 钟摆扫2 ，全防守，双手控制住对方一只胳膊，右手往右侧拉，左手从后面去抓对方肩膀下面像拿背，右手抄对方的腿，然后我的抄腿手同侧的腿用力下压， 转上位
	6. 钟摆扫3 ，全防守，双手控制住对方一只胳膊，右手往右侧拉然后弯曲，左手绕过对方头抓住对手被控制的手，然后右手抓对方裤子，右腿发力下压，转骑乘位
	7. 钟摆扫转十字固 ，手拉住对方后脑按下来，如果对方手在我胸前，那么给对方手肘塞进去压住，一只手压住对方后脖子，如果对方头起来了，那么转为推他的头，拉住他的胳膊， 上腿转下位十字固

	


# 上位半防守

	（控制）
	1. 一定要把对方压平， 否则很容易被翻过来，一定要压住对方另外一侧，可以做一些绞技，也可以把腿抽出来
	
	(攻击手臂)
	1.十字固，我的右手抄对方脖子，先把左腿膝盖逃出来，像骑乘一样切膝，对方身体是侧身，侧着背对我，我的右腿要迈到对方的后脑位置，右手手臂下面是对方的胳膊，做十字固
	2.木村和美国锁，把对方压平之后，可以主要思路攻击手臂，如果对方拿住了我的腋下，可以侧身面向脚的方向，这样对方拿不到我的后背，然后拿住对方手臂进行攻击

	（攻击脖子）	
	棒球绞，需要拿到对方的衣襟给左手，左手从对方脑后穿过，右手把对方的胳膊拿过来压住，然后右手拿棒球绞把位

	（转换）
	转拿背1，把膝盖拿出来，然后切膝，注意切膝的方向，切外侧到骑住对方，看到对方的屁股，前滚翻后蹬对方拿背
	转拿背2，把膝盖拿出来，然后切膝，切膝内侧方向，拿住对方领把，把他拎起来，对方坐起来容易扫我，所以我把对方头拿到我身后，然后转身胯腿迈到对方身后，拿背

	（腿的逃离）
	1. 骑乘，俯下身，压住对方，并且一定要注意压的方向， 方向是我腿被夹住的方向， 把对方的身体压平，对方的肩膀着地， 我可以用肩膀去压对方的喉咙， 然后我可以用手肘去怼对方的腿，让我的腿跑出来， 然后形成上位骑乘
	2. 袈裟，压平对方之后，控制对方领子和手，屁股立起来，然后切膝，重心放在对手身上，蹬对方上面的腿，然后转袈裟
	3. 后仰，俯身下去，压住对方，用被夹住同侧的手搂住对方脖子，然后把能活动的腿拿到另外一侧，我的身体半侧身， 然后抓对方裤子，然后一定要往后挪几步，弄出来空挡防止对方翻我，如果这个时候对方来翻我，我的被夹的腿可以顶起来和抓裤子的手再给对方翻回去，然后活动的腿蹬对方的膝盖，就可以解脱出来，手依然抓住裤子，然后把对方压平才行，然后转侧压， 或者直接再上来转骑乘
	4. 腋下， 先抢腋下，如果拿到腋下，就可以把头放到对方另外一边，崛起屁股进行腿逃离，可以另外一条腿来帮忙，然后切膝，转袈裟
	5. 转身， 转身屁股对着对方脸，我的手抓对方膝盖帮忙抽腿




# 下位半防守

	（控制）
	尽量不要让对方把我压平， 可以把膝盖顶到对手胸部，转z半防守，如果被压平跟被测压一样用手肘顶对方脖子
	
	（扫技）
	扫技1，膝盖要顶到对方的腰部防止他下来压我， 双手用力侧向起身，同时膝盖往前顶，可以转上位
	扫技2，起身弯腰抄对方的右侧大腿，自己的左腿在上面，右腿在下面，左手抓住对方的右脚，左腿往下压，转上位
	扫技3，双手还是抵住对方的领把和袖把，同时塞膝盾，把膝盖划走的同时抢对方腋下，转拿背或者双手抱对方的腿把对方推倒
	扫技4，适合对方上肢搂住我，手抓对手腰带同时膝盖顶起来对方，然后手挡住对方膝盖，扫倒


	(木村)
	1. 对方来压我的时候，我要侧身去做远端的木村， 就是往对手的方向起身，伸胳膊，右手去抓对方的手臂. 一定要侧身，身体朝向右侧，左臂夹住对方的左手，右手抓手腕，木村然后往后倒
	2. 对方压住我，我左手抢腋下，右手过来从下面往上穿，左手抓对方手腕，左腿勾上来圈住对方

	（十字绞）
	对手的右腿被我圈住， 左手虎口向外抓对方的衣领，右手抓住对方的手，然后右手绕过对方后脑把对方带下来，右手抓左手的手肘衣服用力拉

	（拿背）
	抄腋下抓腰带，挪屁股出来，手肘支撑起来，腿往外掰对方的腿，头出来，再把另外一只脚伸进去拿背

	（转换）
	横移出去， 再把腿伸进来转全防守
	把腿伸进去，转蝴蝶防守
	把膝盖顶到对手胸部，转z半防守，可以转全防守， 可以拿背，往上走可以直臂固，可以三角绞
	抱住对手一条大腿，转深度半防守，可以转身到侧压，
	可以抄对方胳膊进行圈臂, 腋下夹住对方手腕，弯曲身体来掰对方手肘



# 浮固

	(攻击手臂)

# 南北位

	（十字固）

	（蟒蛇绞）
	手从对方肩膀穿到另外一侧腋下，抓自己二头肌，然后开始靠近对手，脚勾住对方腿

	（木村）用右手二头肌弯曲抄对方手臂，起身把对方弄侧身，右手抄好对方的手臂，左手抓对方手腕，右手抓自己手腕，按平，重心放到左肩，用力顶右侧

	（南北绞）手穿过对方脖子，然后把身体放低往下走，肩膀压下来，头要贴着对方的头，发力是后背发力

# 拿背

	（策略）
	先搭对方的肩膀，然后是领把，主要进攻脖子，如果脖子防守太好，就抓住胳膊然后转十字固或者反三角

	（裸绞）
	绕过对手脖子，手搭在自己肩膀上，另外一只手伸进去拉自己的手肘，双脚勾住对方的双腿， 顶胯 
	
	（衣襟绞）
	绕过对手脖子，抓对手衣领， 另外一只手抓另外衣领，背部发力，或者另外手伸到对手脖子后面，或者抓对手裤子，然后侧过来，用膝盖顶对手的后脖子

	（片羽绞）
	绕过对手脖子，抓对手衣领，另外一只手从对手的腋下穿上来，然后伸到对方脖子后面往前压迫
	
	（弓箭绞）
	一手抓对手衣领，一手抓裤子，大腿下压，最好两条腿距离很近，类似三角，不行可以抓对方的上面一只手臂进行十字固；也可以外面的一条腿弯过来做反向的三角锁
	
	(转十字固)
	拿背后搭肩的手抓住对方手腕，另外一只手抢对手腋下然后抓自己手腕，搭肩的手绕过头，虾行转十字固。注意这里腿要压住对方，不然对方会起桥。 

	（反三角锁）拿背后搭肩的手抓住对方手腕，另外一只手抢对手腋下然后抓自己手腕，往后蹭一点，把腿搭到对方的胳膊位置，一手抓自己脚踝，把脚要横过来，搭扣，用力的方式是屁股往后坐，身体往前倾
	


# 十字固

	十字固的时候， 对方起身，转三角锁
	十字固的时候， 抓对手的手背，用力按，来掰对方的手腕

	十字固的时候， 对方双手拉在一起，破解办法，1，掰对手胳膊，2，脚来帮忙踹，3，腿过来搭扣压他的胳膊 4，把手肘穿的深一些，然后再拉 5，下面的腿穿过对方胸前到脖子位置，拉起做三角锁 6，把对方远端的手肘抱过来，之后双脚搭扣，拉远端手，对方手就开了，在做近端手十字固

# 橡皮防守

	全防守的时候， 把腿横到对手的脖子后面，然后虾行一点点，（虾行的方向是抬起腿的这个侧面，目的是为了方便把腿搭到对方脖子后面）这个时候对手的一只手已经被我的腿给夹住了，我的一只手反手拉住自己脚踝， 另外一只手要伸到对手的胳膊里面把对手的胳膊给弄出来，防止他不给手伸直，让对手的手不要弯着，对手手直了之后，我们用胳膊和腿把这只手固定住，他就逃不掉了，这个时候可以做如下动作：

	反手贴着对方脖子进入，反手抓自己脚踝，另外一只手过来，进行十字绞
	抓脚踝的手放开，向外推头，把脚丫放到对手下巴位置，1，可以搭腿压他的肩膀，2，可以双手搭扣进行扼颈，腿不同侧的手从脚下穿过去，另外一只手从上面，3，侧身可以转腿木村， 

	橡皮防守的时候，脚勾住对方下巴之后， 可以抓对方另外的手， 蹬手肘， 然后上来做三角锁
	橡皮防守的时候，脚勾住对方下巴之后， 可以控制对方另外一只手， 然后另外一只脚也过来搭下巴，形成双臂锁
	


# 德拉防守

	（转三角锁）
	德拉防守中自己左手控制对手的一只脚， 右手拉衣领，往对方脸的方向拽倒对方，对方双手撑地，左腿蹬对方胯，右腿从对方双手间穿过，拉回来做三角锁

	（转十字固）
	德拉防守中自己左手控制对手的一只脚， 右手拉衣领，往对方脸的方向拽倒对方，对方双手撑地，双腿一起蹬对方胯，把自己的身体贴近对方上身，肩膀着地，侧身，左手抄对方胳膊，左腿和膝盖压对方的肩膀，让对方前滚翻，翻过来之后就可以做十字固了

	（转拿背）
	1. 德拉防守，双手拽对方一只手，屁股尽量挪到侧面，把勾住的脚深入进去，勾对方的另外一侧大腿，形成德拉x，然后把另外一只脚也伸进到对方膝窝处，从后面进行蝴蝶防守，拽腰带可以拿背
	2. 勾左腿深入到对方另外一侧大腿，抓脚踝，右手抓衣领，往我的头上方向带倒对方，当对方倒的时候我的头要往对方腰的位置靠，屁股朝上做菠萝，脚交叉勾住对方的脚，类似x guard，然后抓脚踝的手抓对方的屁股位置的裤子，脚把对方脚和腿带到一边， 同时手带对方腰，把对方带到另外一边拿背

	（扫技）
	1. 椅子扫 ，抓住对手裤脚，腿穿到中间去把对方挑起来，蹬对方跨然后自己起身， 一手抓领，一手高抬对手裤脚， 拉倒对方
	2. 拉远端衣袖，往侧方向倒， 如果不好拉袖子，那么先蹬腿，对方就会用手撑地，这个时候就可以拿到袖子
	3. 双手拉近端的衣袖，然后蹬胯，往头上的方向去摔对方，要紧跟着后滚翻拿上位
	

# X防守

	
	(十字固)
	x guard里面要抓对方袖子然后交给抱腿的手拿住，另外一只手拉衣领下来上腿做十字固
	
	（扫技）
	1. 把对方的脚踝扛到肩膀上，双腿开始推，对方会劈叉，然后手肘撑地起身，对方的腿一直要抗在肩上
	2. 抓住对方的两个袖子，肩一顶就可以起来转上位
	3. 如果对方向后倒，那么一定要快速过腿，否则非常容易进入对方guard，所以尽量让对手往前倒
	
	(拿背)
	把对方的脚踝扛到肩膀上，双腿开始推，然后把对方的脚踝拿到我头的另外一侧，我的手推对方的膝窝，推开之后用我的膝盖顶对方膝窝，手抓腰带转拿背
	

# 蜘蛛防守

	（控制）
	蜘蛛要控制对方跟我的距离， 距离太近的时候， 对方可以卡住我的腿
	蜘蛛在左右转换的时候， 屁股一定要抬起来，这就需要转换的脚要踩一下，然后蹬直腿
	一只脚从手臂下面从外侧伸到内侧，勾住对方腋下，另外一只脚蹬对方手肘
	
	
	（转骑乘扫）
	1. 双腿蹬直对手，然后再突然一松，让对方的重心靠近过来，然后蹬起来对方到头顶，然后后滚翻转骑乘
	2. 抓对方双衣袖，蹬对方手肘，要蹬的很高，破坏对方重心，然后蹬膝盖或剪刀扫倒
	3. 蹬对方胯，蹬对方手肘，拉右手上左腿让对方以为我做三角锁，然后左手抄对方腿，左腿打腋下，右脚蹬对方胯，扫到，因为这里可以拿对方一只手，且脚勾着对方脖子，可以直接对手臂各种各样的进攻，也可以抓自己脚踝和对方手，倒下做三角锁


	（三角锁）
	抓对方双衣袖，蹬对方手肘，腿从上面走，搭腿转三角锁， 对方起身转十字固
	
	（十字固）
	抓对方双衣袖，蹬对方手肘，要蹬的很高，腿从下面走，把对方衣袖往一侧拉扯，然后上腿做十字固


# 开放式防守

	（扫技）
	1. 左脚从手臂下面从外侧伸到内侧，勾住对方腋下，右腿横过来抵住对方胸腹，侧身去抓对方的大腿，然后挑翻对方转上位
	2. 左脚从手臂下面外侧缠绕进来勾住对方腋下，对方的手肘在我的膝窝处，然后勾对方的裆部，右手可以抓衣袖，也可以抄腿，然后把对方往上勾倒
	3. 左脚从手臂下面外侧缠绕进来勾住对方腋下，如果对方半蹲，左脚从腋下绕下来从对方腿穿过去，勾住对方的膝窝，抓衣领往上翻

	（转腿木村）
	1. 左脚从手臂下面从外侧伸到内侧，勾住对方腋下，右腿横过来抵住对方胸腹，换手抓对方衣袖， 然后手拿对方对侧腋下，钻进去然后起身蹬腿转腿木村

	（十字固）
	双腿膝盖向外撑住对方的手肘内侧，双脚蹬对方胯，左腿从手臂下方绕上来，双手用力拉对方手臂同时上腿十字固


# 打乌龟防守

	左手抓对方的后脖子衣领， 右手抓对方裤子，把左腿膝盖塞进去，

	左手穿腋下抓对手左手腕，左脚也挤进去，右手撑地，右小腿压住对方后脖子，左手尽量穿到右大腿上，做十字固

	我抓衣领和对方远端的裤子，膝盖挤进去，往斜后方拉对方，转上位骑乘，对方侧身我就抓对方手腕递给脑后的手抓住，如果对方面对我，我可以直接不松领子的手，另外一只手过来做十字绞，做十字绞一定要把头抵到地面，然后展背

#被全防守

	双手抓住对方的裤子，用手肘去顶对方的大腿内侧，掰开大腿
	双手按住对方的腋下，身体起来，屁股顶高，防止对方抓我的脚踝
	一只手抓住对方衣领，同侧的大腿半蹲，胳膊和大腿夹住对方的一条腿，然后往后靠去掰这条腿

	一只手抓裤子，一只手抵住对方胸，侧身，下压抓裤子的手的手肘，然后把膝盖塞进去，把对方的腿抱起来放到肩上，然后下压，过腿
	双手按住对方腰带，俯身， 屁股后撅顶开对方的腿，然后塞膝盖
	直立上身，一手抓对方衣袖，一只手抵住对方肚子，手臂要直这样对方拉不下去，然后站起来，双脚要尽量的躲在对方屁股后面防止对方拉我的脚，然后起身用手按下去对方的腿，1.跪在对方腿上，切膝过腿转测压, 2. 在按下去的同时直接跳过去
	双手顶住对方腋下，起身， 用手伸进去，然后把对方的腿放到自己肩上过腿




# 被骑乘

	起桥，把对手顶手碰地面，抓手，勾腿，侧翻，起桥，转上位
	往上走，两侧分别推对方大腿快速虾行
	侧身，脚尖勾起对方的踝关节，把另外一条腿膝盖赛过去，转半防守， 再转全防守
	侧身，脚跟勾起对方的腿，把另外一条腿膝盖赛过去，转半防守， 再转全防守
	起桥， 用胳膊顶起对方胯部，然后插入膝盖，然后坐起来， 适合对方比较轻
	当对手的一条腿已经靠前的时候， 起桥，抓对方脚踝，膝盖顶住对方大腿膝窝


# 被测压

	注意被测压的时候双手要紧贴胸前，否则容易被锁，另外靠近对手一条腿要搭起来，防止对方转骑乘
	被搂住脖子的一侧手从腋下穿到前面来，然后另外一只手双手搭扣，一起用手肘顶对方的脖子，把对方顶开后虾行，再进入腿转全防守，或者抄对手腋下钻过去，拿背
	抄腋下，双手搭扣，侧翻
	横向移动，用手肘顶对方脖子，可以用手抄腿，然后勾腿，转半防守
	起桥弄出空间，用手肘怼对手，横向移动，远离对手再进膝盖，转全防守
	起桥， 手肘怼，转身起身，抓对手腿， 然后推倒
	起桥， 手肘怼，转身起身，把脚伸到对手膝窝，准备翻过来
	起桥， 手肘对住对方的腋下，顶起来，然后从腋下钻出去
	
	侧身，上面的手臂去从外侧圈住对方手臂， 另外一只手抓对方手腕，形成木村锁



# 被袈裟固

	手肘推头， 然后身体横向移动， 对手头起来之后， 把腿举上来去勾对方的头
	

# 被拿背

	双手交叉护住自己的胸前，往下坐，一条腿伸直可以解开对方勾腿，从下方逃脱
	手抓住对方手，进行防御， 往后面一个方向倒，蹬腿往上蹭（对方没有搭肩的方向），往一个横向蹭（摆脱腿锁）
	双手抓住对方从肩膀搂住你的胳膊， 然后双手往上拽，头从腋下钻出来
	
	（腿的解脱）
	侧身压住对方的一条腿，然后用我的脚踩住对方的脚踝，然后屁股坐到这个脚踝上，就可以摆脱腿的控制

	（腿的进攻）
	如果对方的腿不正确搭扣，那么把腿横过来搭扣来切
	如果对方的腿像三角锁一样搭到我的腰部，倒向对方搭扣的一侧，然后脚去勾住对方的脚，用手按住对方膝盖，挺胯
	如果对方的腿像三角锁一样搭到我的肋骨，那么我把屁股顶起来，然后脚去勾住对方的脚，用手按住对方膝盖，挺胯

	被站立断头，同侧的手先要防御， 另外手搭住对方脖子钩回来，把对方头带下来，然后把手穿到里面脖子下面，不是穿到对方身体外侧，勾住对方脖子的手臂抓另外手臂二头肌，挺胸降服

	双手反抓对方裤子，双手手肘外旋摆脱蜘蛛， 然后可以斗牛士过腿

# 被扼颈

# 被x防守

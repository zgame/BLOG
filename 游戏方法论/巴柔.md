# 抱腿

	下潜，一个膝盖着地， 抱对方的双腿膝窝，抱双腿
	下潜，一个膝盖着地， 双手抱对方的单腿，可以举起来，摔对方或者勾对方支撑脚； 也可以头靠住对方大腿内侧，双腿夹紧对方腿防止逃脱，肩膀下压
	一手抓对方衣领， 另外一只手抓对方的袖子， 往后撤一步，目的是让对方上步，然后抓袖子的手放开去报对方的单腿

	拉直胳膊，肩顶住对方腋下，顶起，上步同时扫腿
	拉直胳膊，肩顶住对方腋下，顶起，转身半蹲，腰贴抵住对方，拉手摔

	拉衣袖， 大领或搂腰， 转身半蹲撅屁股抵住对手胯部，一脚着地， 一脚要用例去扫对方两条腿
	拉衣袖， 大领或搂腰， 转身一腿半蹲，一腿跪地，撅屁股抵住对手身体，双手拉住对方胳膊，往下拽， 对方摔之后，要用手去盖住对方，侧压
	拉衣袖， 大领或搂腰， 转身半蹲撅屁股抵住对手胯部，一脚着地， 如果对方距离远，一脚扫对方两条腿中间

	双手拉对手的同侧衣领， 用力往下拽， 把对方拉下来， 然后再用其它技巧

# 过腿

	拉住对方脚踝，另外一只手拉住脚背， 往后拽，然后往一面扔
	拉住对方裤脚， 把对方腿拉直， 往一侧扔
	
# 切膝

	用膝盖压住对方的大腿，划过去转侧压

# 上位侧压

	压住，袖车
	衣襟绞，侧压，一手抓后脑领子， 另外一手顺着后脑绕道另外一侧抓领子，近身膝盖压住对手胳膊，用力下压
	棒球绞，侧压，一手拿对方衣襟，包住对手胳膊递给穿后脑的手握住， 然后从对方手下穿过抓棒球把位，近身膝盖压住对手胳膊，用力下压
	侧压做美国锁，不行转身，转十字固
	侧压，对方背对我， 拿住对方胳膊做木村，可以转十字固
	侧压，对方正对我， 拿住对方胳膊往后倒，做木村，

	侧压，做对方美国锁，对方手来帮忙，扳过来对方手，转手臂三角绞
	侧压，起身， 双手搭扣直接绞对方脖子，手穿到对方手臂下面，绕颈， 转手臂三角绞
	侧压，做美国锁，对方手来帮忙， 直接起身做十字固

# 上位骑乘

	抓住对手手腕做木村， 对方另外手过来帮忙就转十字固
	
	上半身压住对方，逐渐往上走， 手和腿把对方的胳膊往上顶，抄胳膊和头，做直臂固，不行可以拉胳膊转十字固

# 十字固

	十字固的时候， 抓对手的手背，用力按，来掰对方的手腕
	十字固的时候， 一个方向不行，转到另外一个方向进行十字固
	十字固的时候， 对方双手拉在一起，三种办法，1，掰对手胳膊，2，脚来帮忙踹，3，腿过来搭扣压他的胳膊

# 全防守

	拉对方下来，对方手靠外， 木村->断头->穿手，搭扣，翻，上位，不松搭扣，腿过来，折颈。
	
	如果对方用双手抓裤子抵住直立，我方抓住对方一只手，另外手抓对方同样这只手的衣袖，拉出来，控制对方一只手
	控制对方手之后可以抄腋下，然后双手搭扣，用头抵住对方手腕，做电话锁，如果失败后面有连招


	
	抓住对方衣领，越深越好，做十字固搭腿，手不放

	抓住对方衣袖，换手，拉起来，头穿过胳膊，用身体压住对方胳膊，手抄住控制住


	下位十字固，拉对方手，一个腿打腋下，侧身，另外一个腿拿上来做十字固， 顶胯
	肩膀十字固，拉对方手，一个腿打腋下，另外一个腿搭到对方肩上，脚搭扣，顶胯
	双臂锁 ， 拉对方双手，两腿夹住对方，一个腿肩上，一个腿在肩膀，顶胯
	三角扣十字固， 拉对方手，一个腿在肩上，一个腿在腋下，脚搭扣，掰对方手臂


	全防守，抱住对方头，拉下来，抄腋下，双手搭扣，解腿扣，虾行为了侧身，电话锁对方胳膊，对方会弯曲胳膊来防守，这时候抓对手手腕往回掰，会硌到对方的前臂肌肉，不能降伏，往外侧掰，对手会失去重心，起桥转上位，如果手依然抓住，可以做美国锁，注意美国锁要拉到靠近对方身体再掰，记得掰对手的二头肌位置，不是前臂

	全防守，抓衣领，衣袖，解扣，虾行，塞膝盖，扫，但是扫失败的时候，把脚插对方腿内做半防守，手抓另外的衣袖，用力拉向一侧，同时膝盖划走，为了拿背，膝盖半防守往外掰，分开对手的腿，这时候勾对方的腿来拿背，如果对方不给腿，就顶跨拉直对方，再勾腿，完成拿背。

	
# 三角锁
	
	拉对方一只手， 腿搭到对方另外一侧肩膀上， 牢牢拉住对方的一只手，自己另外一只手抓搭肩上的脚踝，另外一脚蹬对方胯，转身，上腿搭扣
	三角绞转十字固， 如果对方起身防止三角绞，那么搭扣的腿转一下，转成十字固


# 蜘蛛

# 橡皮防守

	全防守的时候， 把腿横到对手的脖子后面，这个时候对手的一只手已经被我的腿给夹住了，我的一只手反手拉住自己脚踝， 另外一只手要伸到对手的胳膊里面把对手的胳膊给弄出来，防止他不给手伸直，让对手的手不要弯着，对手手直了之后，我们用胳膊和腿把这只手固定住，他就逃不掉了，这个时候可以做如下动作：

# 德拉防守

	德拉防守，双手拽对方一只手，把勾住的脚深入进去，勾对方的另外一侧大腿，然后旋转把另外一只脚也伸进去，从后面进行蝴蝶防守，拽腰带可以拿背

#被全防守

	双手抓住对方的裤子，用手肘去顶对方的大腿内侧，掰开大腿
	双手按住对方的脖子，身体起来，屁股顶高，防止对方抓我的脚踝
	一只手抓住对方衣领，同侧的大腿半蹲，胳膊和大腿夹住对方的一条腿，然后往后靠去掰这条腿

# 被骑乘

	起桥，把对手顶手碰地面，抓手，勾腿，侧翻，起桥，转上位
	往上走，两侧分别推对方大腿快速虾行
	侧身，脚尖勾起对方的踝关节，把另外一条腿膝盖赛过去，转半防守， 再转全防守
	侧身，脚跟勾起对方的腿，把另外一条腿膝盖赛过去，转半防守， 再转全防守
	起桥， 用胳膊顶起对方胯部，然后插入膝盖，然后坐起来， 适合对方比较轻
	当对手的一条腿已经靠前的时候， 起桥，抓对方脚踝，膝盖顶住对方大腿膝窝


# 被测压

	抄腋下，双手搭扣，侧翻
	横向移动，用手肘顶对方脖子，可以用手抄腿，然后勾腿，转半防守
	起桥弄出空间，用手肘怼对手，横向移动，远离对手再进膝盖，转全防守
	起桥， 手肘怼，转身起身，抓对手腿， 然后推倒
	起桥， 手肘怼，转身起身，把脚伸到对手膝窝，准备翻过来
	起桥， 手肘对住对方的腋下，顶起来，然后从腋下钻出去
	
	
	
# 被抱双腿

	赶紧抱住对方的腰， 往后倒，把对方摔过去

# 被抱单腿

	推住对方身体，侧身，抽腿， 抽腿要弯曲膝盖， 抽腿可以向反方向翻滚逃走
	一手勒住脖子，另一手搭扣，掰对方头
	
	脚搭住对方的腿外侧， 贴住对方，手伸进去搭木村，往前翻


# 被断头台

	手搭住对方的脖子，身体撅起来，另外一只手防御

# 被拿背

	手抓住对方手，进行防御， 往后面一个方向倒，蹬腿往上蹭（对方没有搭肩的方向），往一个横向蹭（摆脱腿锁）
	双手抓住对方从肩膀搂住你的胳膊， 然后双手网上拽，头从腋下钻出来
	如果对方的腿是正确搭扣的，那么往直腿的方向横移
	如果对方的腿不正确搭扣，那么在上面腿搭扣


# 被袈裟固

	手肘推头， 然后身体横向移动， 对手头起来之后， 把腿举上来去勾对方的头
	

# 乌龟逃脱（被人南北拿背）

	抱住对方胳膊，翻身，胳膊打开绕到对方后背，转拿背或侧压
	保住对方大腿， 脚蹬地横向转动，转动到对方侧位开始压倒
	
	
	如果对手测位置背书包， 腿支起来屁股，把腿横在对手身后，然后往后倒

# 被蜘蛛

	摆脱一个手，甩对手另外一侧，另外一个手可以做对方十字固


# 扼颈逃脱

	下位，推对手的下巴，横向移动下身到对手成一条线，然后用腿勾住对手的头，把对手勾倒，然后起身，跨到对手身上， 手搭扣，头往下压，电话固攻击对方勒我的胳膊。

	上面的过程中，如果对手也跟着移动，那么就反向勾住对方腿，把对手翻过来。
	

# 被三角绞

	立即挺身， 手抓住被锁的手，防止对方挺胯，然后两只手拉对方的腿
	立即起身， 往下压， 然后双手去锁对方的脖子


# 被十字固

	立即往对手要倒的方向上靠近，压住对方的腿
	立即举腿，后滚翻，翻出去
	
	

	
# 被悬浮固

	横向移动，手抱住对方腿和膝盖，把他绊倒
	抓住对手腿，横向移动出去， 然后再移动回来，把腿放进去
	起桥，然后抓住对手腿，进入半防守
	如果对手拉我衣领，直接电话固对方拉我衣领的胳膊，把对手拉倒

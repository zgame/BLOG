# 抱腿

	下潜，一个膝盖着地， 抱对方的双腿膝窝，抱双腿
	下潜，一个膝盖着地， 双手抱对方的单腿，可以举起来，摔对方或者勾对方支撑脚； 也可以头靠住对方大腿内侧，双腿夹紧对方腿防止逃脱，肩膀下压
	一手抓对方衣领， 另外一只手抓对方的袖子， 往后撤一步，目的是让对方上步，然后抓袖子的手放开去报对方的单腿

# 过腿

	拉住对方脚踝，另外一只手拉住脚背， 往后拽，然后往一面扔
	拉住对方裤脚， 把对方腿拉直， 往一侧扔
	肩膀抵住一条腿，左手反抓对方的衣领，并把左手的手刀放到对手脖子位置，围绕对方的腿绕一下转测压

	手肘抓住对方的脚踝，手搭扣，脚踩住对方的腿，然后肌肉切对方小腿 
	
# 切膝

	用膝盖压住对方的大腿，划过去转侧压
	在切膝的过程中，目的是将对方身体侧过来，注意一定要抢腋下，否则下位的人拿到腋下可以拿背， 抢到腋下之后，肩膀压住对方的肩膀， 头顶住对方的头，再过腿
	在切膝的过程中，腿不能太近，容易被抱腿，  抽腿的时候要转跨，屁股顶住对方，抽腿


# 上位侧压

	压住，袖车
	衣襟绞，测压，左手抓后领，右手拿对方下侧的衣襟交给左手，然后右手从衣襟下面穿过去，切对方的脖子，形成挤压
	十字绞，侧压，一手抓后脑领子， 另外一手顺着后脑绕道另外一侧抓领子，近身膝盖压住对手胳膊，用力下压
	棒球绞，侧压，一手拿对方衣襟，包住对手胳膊递给穿后脑的手握住， 然后从对方手下穿过抓棒球把位，近身膝盖压住对手胳膊，用力下压
	
	侧压做美国锁，不行转身，转十字固
	侧压，对方背对我， 拿住对方胳膊做木村，可以转十字固
	侧压，对方正对我， 拿住对方胳膊往后倒，做木村，

	侧压，做对方美国锁，对方手来帮忙，扳过来对方手，转手臂三角绞
	侧压，起身， 双手搭扣直接绞对方脖子，手穿到对方手臂下面，绕颈， 转手臂三角绞
	侧压，做美国锁，对方手来帮忙， 直接起身做十字固

	

# 袈裟固

	直臂固， 把靠近对手的腿往前伸， 然后靠外的腿跨过对方的头，然后拧身，掰对方的胳膊
	直接从袈裟位，跳对手身上，远侧腿上对方脖子， 做十字固
	


# 上位骑乘

	上位骑乘的腿最好自己搭扣，防止对方腿勾我的腿，可以更稳固

	悬浮固，跪腹部，拉衣领和裤子

	上位骑乘十字固，单膝跪耳测，另外脚半蹲，侧身，抄手，十字固
	单手拉起对方身体，让对方侧身，腿前后固定，抄手，坐肩膀，十字固
	
	一手抱头，一手打开保持平衡，双腿逐渐网上压住对方手臂，一腿贴耳，一腿弯到环绕，抄手，往后顶胯，两侧都可以十字固
	抓住对手手腕做木村， 对方另外手过来帮忙就转十字固
	
	上半身压住对方，逐渐往上走， 手和腿把对方的胳膊往上顶，抄胳膊和头，做直臂固，不行可以拉胳膊转十字固



# 全防守

	拉对方下来，如果对方手靠外， 木村->断头->穿手，搭扣，翻，上位，不松搭扣，腿过来，折颈。
	拉对方下来，如果对方手靠内， 可以扫技，
	如果对方拉不下来，起身抱对方，抓衣领，拉下来，可以做扫技，或者扫技虾行后顶膝的时候，顶高一点顺势转到肩膀上三角扣直接下位降服
	如果对方要逃脱，双手去抓对方的一个衣袖，控制对方的一个手臂，尽量把他手臂拉直， 1.可以拉起来，头穿过胳膊，用身体压住对方胳膊，手抄住控制住 2 可以拉到头的位置，抄腋下做直臂固 3 可以拉到左侧做腿木村或者中途转三角锁 4 可以拉到右侧转拿背



	（转拿背）
	1.一只手抓住对方不同侧手腕， 另外一只手肱二头肌弯曲扣住对方手肘，侧身留出空间，一手拉直对方胳膊并且不要放，另外一只手拉对方后背可以拿背
	2.全防守，抓衣领，衣袖，解扣，虾行，塞膝盖，扫，但是扫失败的时候，把脚插对方腿内做半防守，手抓另外的衣袖，用力拉向一侧，同时膝盖划走，为了拿背，膝盖半防守往外掰，分开对手的腿，这时候勾对方的腿来拿背，如果对方不给腿，就顶跨拉直对方，再勾腿，完成拿背。


	（衣襟绞背后穿攻击脖子）
	1.可以左手拿对方衣襟从后面穿过来，交给右手做棒球握，右手前臂抵到对方的喉咙， 如果对方来推我的右手肘，左手抓住对方衣袖拉过来，左手拉直并且往下推，同时右手用力下拉
	2. 左手拿对方的衣襟从后面穿过来，交给右手做棒球握，双腿下压让对方的头在我的胸前位置，左手交叉去抓衣襟或者对方肩膀，十字绞，如果对方推我的手肘不让我绞，那么转1

	（衣襟绞侧面穿攻击脖子和手臂）
	1.左手拿对方的衣襟，虎口向上握住，衣襟从对方的肱三头肌位置拉过来，同时右手把对方的左手也拉到我的左侧，双腿向上台 为了让对方头部靠近我的胳膊，然后右手从后面绕过对方脖子去抓衣襟，抓住之后可以右手用力拽，左手按住对方胳膊；或者左手也同时在内侧向外推，形成挤压
	2. 左手拿对方的衣襟，虎口向上，衣襟从对方的肱三头肌位置拉过来，同时右手把对方的手也拉到一侧，左手抓衣襟的手不动去控制住对方胳膊，然后上腿做十字固
	

	(下位十字固，攻击手臂)
	1. 双手抓对方的衣袖，把对方的手拉直，对方的手肘在我的腰带处，一只手抓衣袖，另外一只手从对方手上面，我的手下面穿进来搭扣，注意这个时候如果对方想抽走手臂， 他会拉直手臂，这个时候就是上腿的时候，两条腿都要上来，一侧腿要越过头部，做下位十字固
	2. 双手抓对方的衣袖，把对方的手拉直，对方的手肘在我的腰带处，右手抓衣袖，左手从对方手上面，从右手下面穿进来搭扣，如果对方不拉直胳膊，那么我把对方拉下来，把抓住的胳膊交给从后面绕脖子的左手抓住，然后右手抄对方腋下，抄对手的左胳膊，左手也来一起抓住，然后左面侧身，右腿也拿上来，跪到对方的脖子处，开始对对手的左臂进行反向十字固
	3. 抓住对方衣领，越深越好，做十字固搭腿，手不放
	4. 下位十字固，拉对方手，一个腿打腋下，侧身，另外一个腿拿上来做十字固， 顶胯

	（下位臂锁，攻击手臂）
	1. 双手抓对方的衣袖，把对方的手拉直，对方的手肘在我的腰带处，右手抓衣袖，左手从对方手上面，从右手下面穿进来搭扣，如果对方不拉直胳膊，那么我把对方拉下来，把抓住的胳膊交给从后面绕脖子的左手抓住，然后右手抄对方腋下，抄对手的左胳膊，用右手内侧手肘抵住对方手肘，对方的手要卡到我脖子的位置，左手过来帮忙拉我的右手肘，做直臂固
	2. 如果对方用双手抓裤子抵住直立，我方抓住对方一只手，另外手抓对方同样这只手的衣袖，拉出来，控制对方一只手，控制对方手之后可以抄腋下，然后双手搭扣，用头抵住对方手腕，做直臂固
	3. 全防守，抱住对方头，拉下来，抄腋下，双手搭扣，解腿扣，虾行为了侧身，电话锁对方胳膊，对方会弯曲胳膊来防守，这时候抓对手手腕往回掰，会硌到对方的前臂肌肉，不能降伏，往外侧掰，对手会失去重心，起桥转上位，如果手依然抓住，可以做美国锁，注意美国锁要拉到靠近对方身体再掰，记得掰对手的二头肌位置，不是前臂


	（下位十字绞，攻击脖子）
	1. 左手抄对手的手肘，从手肘下面穿过去，抓对方远端的衣领，右手抓另一侧已领，右手往前顶，左手往回拉，做十字绞
	2. 抓左侧，对方右侧的衣领，左手在上，右手在下，左手发力往左下带，对方的头下来之后，手腕转动绕过对方头，注意把位不能变，绕过之后就是十字绞
	3. 抓左侧，对方右侧的衣领， 右手在上，左手在下，左手发力，往左下带，对方的头下来之后，左手去抓对方的腰带，左手发力用胳膊推对方头，右手拉，形成挤压


	（下位三角锁，攻击脖子）
	1. 双手各自抓对方的衣袖，右手推，左手拉，同时挺胯，上右腿到对方肩膀上，左手反方向拉住对方胳膊，向右侧身，把右脚放到对方脖子后面，左脚蹬对方胯或者地面，有一定空间再搭扣，双腿膝盖加紧，挺胯
	2. 如果双脚已经上来，对方手藏起来不给拉手， 用双手在对方背后搭扣，然后双腿下压，双手勒紧
	3. 如果双脚已经上来，对方手藏起来不给拉手， 那么把三角锁反向，用左手抵住对方的胳膊肘，左腿被右腿压住搭扣，用力


	（肩膀三角扣，攻击手臂）
	肩膀十字固，拉对方手，一个腿打腋下，另外一个腿搭到对方肩上，脚搭扣，顶胯
	双臂锁 ， 拉对方双手，两腿夹住对方，一个腿肩上，一个腿在肩膀，顶胯
	三角扣十字固， 拉对方手，一个腿在肩上，一个腿在腋下，脚搭扣，往一侧掰对方手臂
	肩锁，横向推对方的手肘，当手肘超过中线之后，开始双手掰对手的胳膊
	木村，对方的手背过去藏起来，不转腿木村，而是双手搭扣开始压按对方的肘关节
	压脖子，直接抓对方的衣领，然后用拳峰去挤压对方的脖子
	直臂固，抄腋下，然后做直臂固
	肩甲固，左手抓住对方左手腕，右手从对方手肘中穿过并按住自己的左大腿，把对方的手藏入腿缝，把左腿拿到对方肩上来，右手搭住大腿，另外一只手掰对方手肘；或者右腿蹬对方胯，同时右手掰对手的手肘



# 上位半防守

	一定要把对方压平， 否则很容易被翻过来，一定要压住对方另外一侧，可以做一些绞技，也可以把腿抽出来
	可以做直臂固或十字固
	手臂三角绞
	木村和美国锁
	棒球绞

	俯下身，压住对方，并且一定要注意压的方向， 方向是我腿被夹住的方向， 把对方的身体压平，对方的肩膀着地， 我可以用肩膀去压对方的喉咙， 然后我可以用手肘去怼对方的腿，让我的腿跑出来， 然后形成上位骑乘

	俯身下去，压住对方，用被夹住同侧的手搂住对方脖子，然后把能活动的腿拿到另外一侧，我的身体半侧身， 然后抓对方裤子，然后一定要往后挪几步，弄出来空挡防止对方翻我，如果这个时候对方来翻我，我的被夹的腿可以顶起来和抓裤子的手再给对方翻回去，然后活动的腿蹬对方的膝盖，就可以解脱出来，手依然抓住裤子，然后把对方压平才行，然后转侧压




# 十字固

# 三角锁
	
	拉对方一只手， 腿搭到对方另外一侧肩膀上， 牢牢拉住对方的一只手，自己另外一只手抓搭肩上的脚踝，另外一脚蹬对方胯，转身，上腿搭扣
	三角绞转十字固， 如果对方起身防止三角绞，那么搭扣的腿转一下，转成十字固


# 蜘蛛

	蜘蛛在左右转换的时候， 屁股一定要抬起来，这就需要转换的脚要踩一下，然后蹬直腿

	蜘蛛可以蹬直对手， 然后再一松，让对方的重心靠近过来，然后蹬起来对方到头顶，然后后滚翻转骑乘

# 橡皮防守

	全防守的时候， 把腿横到对手的脖子后面，然后虾行一点点，（虾行的方向是抬起腿的这个侧面，目的是为了方便把腿搭到对方脖子后面）这个时候对手的一只手已经被我的腿给夹住了，我的一只手反手拉住自己脚踝， 另外一只手要伸到对手的胳膊里面把对手的胳膊给弄出来，防止他不给手伸直，让对手的手不要弯着，对手手直了之后，我们用胳膊和腿把这只手固定住，他就逃不掉了，这个时候可以做如下动作：

	反手贴着对方脖子进入，反手抓自己脚踝，另外一只手过来，进行十字绞
	抓脚踝的手放开，向外推头，把脚丫放到对手下巴位置，1，可以搭腿压他的肩膀，2，可以双手搭扣进行扼颈，腿不同侧的手从脚下穿过去，另外一只手从上面，3，侧身可以转腿木村， 

	橡皮防守的时候，脚勾住对方下巴之后， 可以抓对方另外的手， 蹬手肘， 然后上来做三角锁
	橡皮防守的时候，脚勾住对方下巴之后， 可以控制对方另外一只手， 然后另外一只脚也过来搭下巴，形成双臂锁
	


# 德拉防守

	德拉防守中自己手控制对手的一只脚， 另外一只手拉衣领，往一侧拽倒对方，蹬跨，穿腿，做三角锁
	德拉防守中自己手控制对手的一只脚， 另外一只手拉衣领，往一侧拽倒对方，蹬跨，侧身，双手抄对方胳膊，腿和膝盖压对方的肩膀，旋转，带着对方前滚翻，翻过来之后就可以做十字固了

	德拉防守，双手拽对方一只手，把勾住的脚深入进去，勾对方的另外一侧大腿，然后旋转把另外一只脚也伸进去，从后面进行蝴蝶防守，拽腰带可以拿背

	德拉防守，抓住对手裤脚，腿穿到中间去把对方挑起来，蹬对方跨然后自己起身， 一手抓领，一手高抬对手裤脚， 拉倒对方，

#被全防守

	双手抓住对方的裤子，用手肘去顶对方的大腿内侧，掰开大腿
	双手按住对方的腋下，身体起来，屁股顶高，防止对方抓我的脚踝
	一只手抓住对方衣领，同侧的大腿半蹲，胳膊和大腿夹住对方的一条腿，然后往后靠去掰这条腿

	一只手抓裤子，一只手抵住对方胸，侧身，下压抓裤子的手的手肘，然后把膝盖塞进去，把对方的腿抱起来放到肩上，然后下压，过腿
	双手按住对方腰带，俯身， 屁股后撅顶开对方的腿，然后塞膝盖
	直立上身，一手抓对方衣袖，一只手抵住对方肚子，手臂要直这样对方拉不下去，然后站起来，双脚要尽量的躲在对方屁股后面防止对方拉我的脚，然后起身用手按下去对方的腿，1.跪在对方腿上，切膝过腿转测压, 2. 在按下去的同时直接跳过去
	双手顶住对方腋下，起身， 用手伸进去，然后把对方的腿放到自己肩上过腿



# 下位半防守

	尽量不要让对方把我压平， 可以把膝盖顶到对手胸部，转z半防守，如果被压平跟被测压一样用手肘顶对方脖子
	
	扫技，膝盖要顶到对方的腰部防止他下来压我， 双手用力侧向起身，同时膝盖往前顶，可以转上位
	扫技2，起身弯腰抄对方的右侧大腿，自己的左腿在上面，右腿在下面，左手抓住对方的右脚，左腿往下压，转上位

	木村，对方来压我的时候，我要侧身去做远端的木村， 就是往对手的方向起身，伸胳膊，右手去抓对方的手臂
	十字绞，对手的右腿被我圈住， 左手虎口向外抓对方的衣领，右手抓住对方的手，然后右手绕过对方后脑把对方带下来，右手抓左手的手肘衣服用力拉

	横移出去， 再把腿伸进来转全防守
	把腿伸进去，转蝴蝶防守
	把膝盖顶到对手胸部，转z半防守，可以转全防守， 可以拿背，往上走可以直臂固，可以三角绞
	抱住对手一条大腿，转深度半防守，可以转身到侧压，
	可以抄对方胳膊进行圈臂,腋下夹住对方手腕，弯曲身体来掰对方手肘


# 被骑乘

	起桥，把对手顶手碰地面，抓手，勾腿，侧翻，起桥，转上位
	往上走，两侧分别推对方大腿快速虾行
	侧身，脚尖勾起对方的踝关节，把另外一条腿膝盖赛过去，转半防守， 再转全防守
	侧身，脚跟勾起对方的腿，把另外一条腿膝盖赛过去，转半防守， 再转全防守
	起桥， 用胳膊顶起对方胯部，然后插入膝盖，然后坐起来， 适合对方比较轻
	当对手的一条腿已经靠前的时候， 起桥，抓对方脚踝，膝盖顶住对方大腿膝窝


# 被测压

	注意被测压的时候双手要紧贴胸前，否则容易被锁，另外靠近对手一条腿要搭起来，防止对方转骑乘
	被搂住脖子的一侧手从腋下穿到前面来，然后另外一只手双手搭扣，一起用手肘顶对方的脖子，把对方顶开后虾行，再进入腿转全防守，或者抄对手腋下钻过去，拿背
	抄腋下，双手搭扣，侧翻
	横向移动，用手肘顶对方脖子，可以用手抄腿，然后勾腿，转半防守
	起桥弄出空间，用手肘怼对手，横向移动，远离对手再进膝盖，转全防守
	起桥， 手肘怼，转身起身，抓对手腿， 然后推倒
	起桥， 手肘怼，转身起身，把脚伸到对手膝窝，准备翻过来
	起桥， 手肘对住对方的腋下，顶起来，然后从腋下钻出去
	
	
	
# 被抱双腿

	赶紧抱住对方的腰， 往后倒，把对方摔过去

# 被抱单腿

	推住对方身体，侧身，抽腿， 抽腿要弯曲膝盖， 抽腿可以向反方向翻滚逃走
	一手勒住脖子，另一手搭扣，掰对方头
	
	脚搭住对方的腿外侧， 贴住对方，手伸进去搭木村，往前翻


# 被断头台

	手搭住对方的脖子，身体撅起来，另外一只手防御
	被站立断头，同侧的手先要防御， 另外手搭住对方脖子和后背，可以抓住， 绕到对手侧身位置，膝盖顶对方膝窝，把对方顶倒， 转侧压，然后主要用肩的力量去压对手的脖子
	

# 被拿背

	双手交叉护住自己的胸前，往下坐，一条腿伸直可以解开对方勾腿，从下方逃脱
	手抓住对方手，进行防御， 往后面一个方向倒，蹬腿往上蹭（对方没有搭肩的方向），往一个横向蹭（摆脱腿锁）
	双手抓住对方从肩膀搂住你的胳膊， 然后双手网上拽，头从腋下钻出来
	如果对方的腿是正确搭扣的，那么往直腿的方向横移
	如果对方的腿不正确搭扣，那么在上面腿搭扣


# 被袈裟固

	手肘推头， 然后身体横向移动， 对手头起来之后， 把腿举上来去勾对方的头
	

# 乌龟逃脱（被人南北拿背）

	抱住对方胳膊，翻身，胳膊打开绕到对方后背，转拿背或侧压
	保住对方大腿， 脚蹬地横向转动，转动到对方侧位开始压倒
	
	
	如果对手测位置背书包， 腿支起来屁股，把腿横在对手身后，然后往后倒

# 被蜘蛛

	反手抓对方手，横移，然后用脚踩对方大腿内侧，摆脱一个手，甩对手另外一侧，另外一个手可以做对方十字固
	横移，脚踩对方大腿，手挣脱，另外一只手绕一下，甩开对方的腿，另一只脚迈过去，然后侧压
	左手反抓对方衣袖，双手手肘内旋摆脱蜘蛛， 右手抓左手位置的裤脚，右手一拉对方身体就侧过来了，膝盖压上去
	双手反抓对方裤子，双手手肘内旋摆脱蜘蛛， 然后可以往一侧甩过腿。


# 被扼颈

	下位，推对手的下巴，横向移动下身到对手成一条线，然后用腿勾住对手的头，把对手勾倒，然后起身，跨到对手身上， 手搭扣，头往下压，电话固攻击对方勒我的胳膊。

	上面的过程中，如果对手也跟着移动，那么就反向勾住对方腿，把对手翻过来。
	

# 被三角绞

	立即挺身， 手抓住被锁的手，防止对方挺胯，然后两只手拉对方的腿
	立即起身， 往下压， 然后双手去锁对方的脖子


# 被十字固

	立即往对手要倒的方向上靠近，压住对方的腿
	立即举腿，后滚翻，翻出去
	
	

	
# 被悬浮固

	抓对手远处的腿，把对方撂倒
	横向移动，手抱住对方腿和膝盖，把他绊倒
	抓住对手腿，横向移动出去， 然后再移动回来，把腿放进去
	起桥，然后抓住对手腿，进入半防守或者深防守
	如果对手拉我衣领，直接电话固对方拉我衣领的胳膊，把对手拉倒
